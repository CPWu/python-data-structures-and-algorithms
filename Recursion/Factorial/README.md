# Factorial

Goal: Use Python to implement Factorials

## Overview

In mathematics, the factorial of a non-negative integer n, denoted by n!, is the product of all positive integers less than or equal to n:

n! = n·(n−1)·(n−2)...3·2·1

For example,
5! = 5·4·3·2·1 = 120
